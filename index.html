Estrutura básica do HTML

Voltar ao inicio

Head

Este elemendo faz parte da estrutura básica do HTML, o mesmo possui as meta informações do HTML

Body

Este elemento definirá o corpo do documento, nele irá constar as informações visuais para o usuário como os cabelhaços, imagens, links e etc

Atributos das tags

Voltar ao inicio

Os atributos são informações adicionais que podem ser adicionadas as tags.

  • Todos os elementos podem possuir atributos
  • Atributos adicionam informações extras aos elementos
  • Atributos são adicionados na tag inicial
  • Atributos geralmente possuem um nome e depois o valor, por exemplo: name="value"

Além dos cabeçalhos possuímos outras tags como o "p" esta utilizada para digitar seus paragráfos, como este que você lê agora. Temos também tags que podem modificar a forma como você enxerga o texto, por exemplo para deixar em negrito uma palavra usamos strong. Possuímos também outras tags, por exemplo o "i", é utilizado para deixar uma palavra em itálico. Temos diversas outras tags que podem ser utilizadas, eu consigo marcar um texto, consigo também fazer ficar visual um número x elevado a y

Listas

Voltar ao inicio

Listas ordenadas

A lista ordenada, inclui automaticamente uma numeração antes do texto

Lista não ordenada

A lista não ordenada, não inclui a numeração, fazendo assim sentido entre a diferença de nomes, em vez disso ela adiciona uma bola antes do texto
